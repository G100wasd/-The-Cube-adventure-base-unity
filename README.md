# G2 - 2D平台动作游戏

基于Unity 2022.3开发的2D平台动作游戏，包含完整的游戏循环、技能系统、背包管理和WebGL构建支持。

## 🎮 游戏概述

G2是一个2D平台动作游戏，玩家可以控制角色在平台上跳跃、战斗，使用各种技能击败敌人并收集物品。游戏采用MVC架构设计，具备完整的游戏状态管理和UI系统。

### 核心特性
- **玩家控制**：灵活的移动、跳跃和多种技能
- **敌人AI**：随机移动、自动转向和受击反馈
- **技能系统**：5种独特技能（冲刺、恢复、超级跳、强化攻击、射击）
- **背包管理**：物品收集、使用和显示
- **掉落系统**：权重随机掉落机制
- **状态管理**：开始/游戏状态切换、暂停系统

## 🛠 技术栈

- **引擎**: Unity 2022.3.62f2c1 (2D项目模板)
- **编程语言**: C# (62个脚本文件)
- **主要插件**:
  - DOTween (动画系统)
  - TextMesh Pro (高级文本渲染)
- **架构模式**: MVC模式、单例模式、事件驱动
- **构建目标**: WebGL (960×600分辨率)

## 🚀 如何运行

### 在Unity编辑器中运行
1. 使用Unity 2022.3或更高版本打开项目
2. 打开 `Assets/Scenes/GameScene.unity`
3. 点击Play按钮开始游戏

### WebGL版本
项目已预构建WebGL版本，位于 `WebGL/` 目录：
1. 打开 `WebGL/index.html` 在浏览器中运行
2. 或使用本地HTTP服务器（如Python的 `python -m http.server`）

### 重新构建WebGL
1. 在Unity编辑器中：File → Build Settings
2. 选择WebGL平台
3. 点击Build，选择输出目录

## 🎯 游戏控制

### 基本控制
- **A/D 或 左右箭头**：水平移动
- **空格键**：跳跃
- **鼠标**：瞄准方向

### 技能快捷键（数字键1-5）
1. **冲刺** (Dash) - 快速水平冲刺
2. **恢复** (Restore) - 恢复生命值
3. **超级跳** (Super Jump) - 更高的跳跃
4. **强化攻击** (Power Hit) - 暂时增强攻击力
5. **射击** (Shoot) - 向鼠标方向射击

### 技能解锁
- 通过碰撞场景中的技能触发器解锁新技能
- 每个技能对应一个ID，解锁后可在背包中使用

## 📦 背包系统

- 收集敌人掉落的物品
- 物品分为6种类型：
  0. 冲刺道具
  1. 恢复道具
  2. 超级跳道具
  3. 强化攻击道具
  4. 射击道具
- 使用数字键触发对应技能，自动消耗背包中的道具

## 🔧 开发工具

### GM命令（编辑器菜单）
Unity编辑器顶部菜单栏提供GM命令工具：
- **GMcmd/读取表格**：输出PackageTable数据
- **GMcmd/读取背包**：显示当前背包物品
- **GMcmd/添加背包**：添加30个冲刺道具
- **GMcmd/添加ME道具**：添加1个ME道具

### 技能权重系统
敌人掉落采用动态权重算法：
- 初始权重：已解锁技能7:3:0:0:0
- 最终权重：所有已解锁技能平均分布
- 权重随解锁技能数量动态调整
